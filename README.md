## 💻 기능구현방식
### 첫번째화면_무 값 입력
- TextField를 이용한 무 값 입력
- UserDefaults를 이용한 무 값 저장 기능 
- Google AdMob을 이용한 배너 광고

### 두번째화면_이번주 무 값 계산 
- URLSession을 이용해 TurnipService에서 Networking
- chart 라이브러리를 이용해 이번주 무 값 그래프 그리기
